  • О сайте

BnSbase.com — проект энтузиастов по созданию лучшей базы знаний по blade and soul в интернете.

Начавшись как просто страничка с деревом морфа, сайт вырос в большое количество понастоящему нужных статей.

Все статьи актуальны для сервера PlayBNS.

При копировании любых материалов ссылка на этот сайт обязательна.
